# WhiteLine — Landing Page

Premium IT kompaniya uchun React (Vite) landing page. Oq + ko'k gradient dizayn tizimi, zamonaviy UI/UX trendlari (glassmorphism, gradient blobs, scroll-reveal animatsiyalar, marquee).

## Ishga tushirish

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # production build -> dist/
npm run preview  # build'ni ko'rish
```

## Tuzilma

- `src/components/` — Navbar, Hero (CEO Temur Rakhimovich), Marquee, Services, Portfolio, Leadership, CTA, Footer
- `src/data.js` — mijozlar (logolar + URL'lar + qilingan ishlar) va xizmatlar
- `public/assets/` — CEO surati va mijoz logolari

## Mijozlar / Portfolio
Uztelecom · BRB · Ibrat Farzandlari · Elektron Kooperatsiya Portali · Calora · Andeli
(har biri logo, kategoriya, qilingan ish tavsifi va clickable URL bilan)
